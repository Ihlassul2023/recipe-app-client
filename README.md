## üíª How To Install?

- Clone This Repo

```
git clone https://github.com/Ihlassul2023/recipe-app-client
```

- Go To Folder Repo

```
cd recipe-app-client
```

- Install Module

```
npm install
```

- Type ` npm run dev` To Start Website
